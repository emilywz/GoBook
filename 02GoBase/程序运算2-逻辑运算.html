逻辑运算

逻辑运算

概述

  • 逻辑运算结果true或false
  • 运算符包括相等、与、或、非：==，&&，||，！
  • 条件与：同时为真就为真
  • 条件或：只要有一个为真，结果就为真
  • 条件非：对结果取反
func main() {

    //a1,b1为true
    // a0,b0为false
    var a1 = (5 > 3)
    var a0 = (5 < 3)
    var b1 = (5 == (2 + 3))
    var b0 = (5 != (2 + 3))

    //true true
    fmt.Println(a1, b1)
    //false false
    fmt.Println(a0, b0)

    //条件与：同时为真就为真
    //true
    fmt.Println(a1 && b1)
    //false
    fmt.Println(a1 && b0)

    //条件或：只要有一个为真，结果就为真
    //true
    fmt.Println(a1 || b1)
    //true
    fmt.Println(a1 || b0)
    //false
    fmt.Println(a0 || b0)

    //条件非：对结果取反
    //false false
    fmt.Println(!a1, !b1)
    //true true
    fmt.Println(!a0, !b0)
}
